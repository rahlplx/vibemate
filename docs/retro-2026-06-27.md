## What Didn't Go Well

### 1. State Drift (HAD)
- **Problem:** `state.json` was stale (484 tests → actual 889)
- **Impact:** Confusion about current state
- **Fix:** Auto-update state after every harness pass

### 2. File Lock Issues (WINDOWS)
- **Problem:** `better-sqlite3` EBUSY on Windows, cloned repos in project root
- **Impact:** 30+ minutes debugging
- **Fix:** Move cloned repos to temp dir, use `.bunignore` (but bun ignores it)

### 3. Regex `g` Flag Persistence (BUG)
- **Problem:** `scanForSecrets` reused regex with `g` flag, `lastIndex` persisted
- **Impact:** AWS key detection failed silently
- **Fix:** Create `new RegExp()` per call, not per pattern

### 4. Placeholder False Positives (BUG)
- **Problem:** "EXAMPLE" in AWS key triggered placeholder filter
- **Impact:** Enterprise-grade keys missed
- **Fix:** Anchor placeholder patterns with `^...$`

### 5. Template Literals in PowerShell (ENV)
- **Problem:** `bun -e` with template literals failed on Windows
- **Impact:** Simulation loop couldn't run inline
- **Fix:** Write to `.ts` file instead
