# /vibe:learn — Self-Improvement Engine

## Purpose
Learn from experience and improve future performance.

## Input
- Retro feedback
- OKF bundle
- Evolution history

## Process
1. Analyze retro feedback for patterns
2. Update skill effectiveness scores
3. Generate new rules (if needed)
4. Refine question bank
5. Optimize decision matrices
6. Update OKF bundle

## Output
- Updated OKF bundle
- Updated `.vibe/evolution.json`
- New rules (if generated)

## Quality Gates
- [ ] Patterns identified
- [ ] Rules are actionable
- [ ] OKF bundle updated
- [ ] No regressions introduced
