# learn-execute

Execute the TDD workflow plan generated by learn-plan.

## Trigger

When user says "execute plan", "run TDD", "implement fixes", "start building", or after learn-plan completes.

## Workflow

1. Read the spec plan from learn-plan output
2. For each slice (by priority):
   a. For each task in the slice:
      - If type=tdd: write failing test first (RED)
      - If type=implement: implement fix to pass test (GREEN)
      - If type=test: verify no regressions
      - If type=docs: update documentation
   b. Run full test suite after each task
   c. Commit after each task
3. Track progress: completed tasks / total tasks
4. Report: pass/fail for each task, regressions found

## Output

```json
{
  "progress": {
    "totalTasks": 24,
    "completedTasks": 12,
    "passRate": 1.0,
    "regressions": 0
  },
  "completedSlices": ["security", "architecture"],
  "remainingSlices": ["testing", "tooling"]
}
```

## TDD Cycle

```
RED → GREEN → REFACTOR → COMMIT
  ↓       ↓         ↓        ↓
failing  passing   clean    saved
test     test      code     state
```
