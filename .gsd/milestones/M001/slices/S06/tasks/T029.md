# T029: Auth Module — Token CRUD and Storage

## Description
Implement `src/mcp/auth.ts` with token management (store, retrieve, validate, revoke).

## Acceptance Criteria
- Token can be stored (in-memory + optional file persistence)
- Token can be retrieved by key
- Token can be validated (check expiry, signature format)
- Token can be revoked/deleted
- Token type: `{ token: string; expiresAt?: number; tier: 'free' | 'pro' | 'team' | 'enterprise'; userId?: string }`

## Dependencies
- None
