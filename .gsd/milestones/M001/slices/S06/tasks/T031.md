# T031: Auth Middleware — Token Validation on Tool Calls

## Description
Implement middleware that checks auth token before executing paid skill calls.

## Acceptance Criteria
- Reads token from env var (VIBEMATE_TOKEN) or config
- Validates token expiry
- Determines user tier
- Passes tier info to tool handlers
- Returns auth error for expired/missing tokens on paid skills
- Free tier skills (spec generator) work without auth
