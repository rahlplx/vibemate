# T032: Auth CLI Commands — Login, Logout, Status

## Description
Add `vibemate auth login`, `vibemate auth logout`, `vibemate auth status` CLI commands.

## Acceptance Criteria
- `vibemate auth login` opens browser for OAuth and stores token
- `vibemate auth logout` removes stored token
- `vibemate auth status` shows current auth state (logged in/out, tier, expiry)
- Non-interactive mode: `--token` flag for CI/CD
