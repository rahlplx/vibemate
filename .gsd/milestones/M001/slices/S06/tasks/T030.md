# T030: Auth OAuth Flow — URL Generation and Browser Launch

## Description
Implement OAuth flow: generate vibemate.dev OAuth URL, open browser, receive callback.

## Acceptance Criteria
- Generates OAuth URL with client_id, redirect_uri, state, scope
- Opens browser to OAuth URL
- Starts local HTTP server for callback
- Receives authorization code and exchanges for token
- Stores token via Auth module
