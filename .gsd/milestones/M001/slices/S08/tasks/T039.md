# T039: Auto-Fix — MCP Tool Registration + CLI Integration

## Description
Register auto-fix as MCP tool `vibemate_fix` and add CLI commands.

## Acceptance Criteria
- Tool registered as `vibemate_fix`
- Parameters: `{ scan?: boolean; fix?: boolean; dryRun?: boolean; issues?: string[] }`
- CLI: `vibemate fix scan`, `vibemate fix apply`, `vibemate fix status`
- Dry-run shows preview before applying
- Returns structured fix report
