# T038: Auto-Fix — Fix Application and Rollback

## Description
Implement fix application logic with backup and rollback support.

## Acceptance Criteria
- Creates backup before applying fixes
- Applies fixes: writes files, updates configs, installs deps
- Rollback support: restores from backup on fix failure
- Returns fix result: `{ fixed: boolean; file: string; backup?: string; error?: string }`
- Dry-run mode: shows what would be fixed without applying
