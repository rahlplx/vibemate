# T037: Auto-Fix Module — Issue Detection and Categorization

## Description
Implement `src/mcp/tools/auto-fix.ts` with issue detection logic for common project problems.

## Acceptance Criteria
- Detects: missing env files, uncommitted changes, outdated deps, missing configs
- Each issue has: `type`, `severity` (critical/high/medium/low), `description`, `fix`, `file`
- Categories: `config`, `dependency`, `security`, `project`
- Scan project root for issues
- Returns list of detected issues
