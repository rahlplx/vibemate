# T034: Auto-Complete Module — Schema and Suggestion Types

## Description
Implement `src/mcp/tools/auto-complete.ts` with suggestion schema and types.

## Acceptance Criteria
- Suggestion types: `spec_section`, `tech_stack`, `feature`, `risk_flag`
- Each suggestion has: `type`, `label`, `description`, `confidence`, `action`
- Zod schema for validation
- SuggestionRequest: `{ query: string; context?: { idea?: string; stack?: string } }`
- SuggestionResponse: `{ suggestions: Suggestion[]; query: string }`
