# T035: Auto-Complete — Suggestion Generation Logic

## Description
Implement suggestion generation: pattern matching on partial input to suggest completions for spec sections, features, and risk flags.

## Acceptance Criteria
- Suggests spec sections based on idea keywords (e.g., "auth app" → suggests Auth sections)
- Suggests tech stack options based on partial stack name
- Suggests risk flags based on idea domain (e.g., "payment" → PCI compliance)
- Returns top 5 suggestions ranked by confidence
- Handles empty query (returns trending/featured suggestions)
